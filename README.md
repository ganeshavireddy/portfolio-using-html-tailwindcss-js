# Durga Venkata Ganesh | Personal Portfolio 🚀

**🌐 Live Demo:** [https://portfolio-using-html-tailwindcss-js.vercel.app/](https://portfolio-using-html-tailwindcss-js.vercel.app/)

A modern, high-performance personal portfolio built to showcase my projects, skills, and experience as a Full-Stack Software Engineer. Designed with a sleek dark theme, glassmorphism UI, and smooth scroll animations.

## ✨ Features

* **Dark Mode & Glassmorphism:** A premium, modern UI featuring frosted-glass elements and deep contrast.
* **Smooth Scroll Animations:** Dynamic section reveals powered by the Intersection Observer API.
* **Fully Responsive:** Fluid layouts that look perfect on desktops, tablets, and mobile devices.
* **Project Showcase:** Dedicated interactive cards for full-stack and data analytics projects.
* **Resume Integration:** Direct link to view/download the latest CV.
* **Custom Contact UI:** A sleek, "Digital Architect" themed contact form.

## 🛠️ Tech Stack

This project is built using vanilla web technologies to remain lightweight and fast, without the need for a heavy build process.

* **HTML5** - Semantic structure
* **CSS3** - Custom animations and variables
* **JavaScript (Vanilla)** - DOM manipulation and scroll observers
* **Tailwind CSS (via CDN)** - Rapid utility-class styling
* **Google Fonts** - *Plus Jakarta Sans* (Headings) & *Inter* (Body)
* **Material Symbols** - Google's modern icon library
